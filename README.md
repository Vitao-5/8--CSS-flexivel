# CSS Flexível com SASS

O projeto foi refatorado para usar SASS com estrutura modular em partials.

- `scss/`: Arquivos SASS fonte
  - `_base.scss`: Estilos base (reset, body, etc.)
  - `_variaveis.scss`: Variáveis SASS
  - `_mixins.scss`: Mixins reutilizáveis
  - `_layout.scss`: Estrutura da página (header, main, footer)
  - `_componentes.scss`: Componentes (botões, cards, etc.)
  - `_responsive.scss`: Media queries
  - `estilos.scss`: Arquivo principal que importa os partials
